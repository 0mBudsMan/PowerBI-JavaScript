Custom Layout Showcase


Use this showcase to learn how to leverage the custom layout API for a dynamic embedding of visuals

 1. Select the visuals from the menu on the left
The configuration of each visual will change between 'Hide'/ 'show', and the app will calculate the position of each visual (see code)

 2. Change the layout to fit different screens
By choosing different layouts, the application sets size and position of each visual (see code)
• 3 columns: usually fits a Desktop screen
• 2 columns: usually fits a Tablet or small laptop screens
• 1 columns: fits a mobile screen
