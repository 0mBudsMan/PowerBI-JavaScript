Custom Layout Demo


Use this demo to learn how to leverage the custom layout API.
The Embedded object is a report, which is used as a visuals container.
By selecting or un-selecting the visual name on the left, you change the configuration of each visual between 'Hide/Show'.
The application calculates the position of each visual according to the selected number of columns and visuals.
In this demo, you can change the layout of your visuals to fit into different screens:
 - 3 columns: usually fits a Desktop screen
 - 2 columns: usually fits a Tablet or small laptop screens
 - 1 columns: fits a mobile screen
By choosing different layouts, the code actually changes the configuration of each visual, by setting its size and position.
You can see and use the code for all of the above through GitHub. Just click "Get the code".
