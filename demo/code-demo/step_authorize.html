Embed Your Own Report

Prerequisites

  • A workspace in Azure PowerBI Embedded Service.
  • PowerBI Report imported to your workspace.

Flow

Once you have a report imported to PowerBI Workspace, you are ready to embed it.
To Embed a report you need to get an Embed-Token. You can create this token in multiple ways:
  • Using PowerBI-Cli tool.
  • In .Net Code using Microsoft.PowerBI.Core package.
  • In NodeJS code using powerbi-api package.
