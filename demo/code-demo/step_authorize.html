Prerequisites

  • A workspace in the Power BI embedded service.
  • Power BI report imported to your workspace.

Instructions to generate an embed App Token

Once you have improted a report into Power BI workspace, you are ready to embed it.

To embed a report, you need to get an embed App Token. You can create this token in multiple ways.
  • Using PowerBI-Cli tool.
  • From .Net Code using Microsoft.PowerBI.Core package.
  • From NodeJS code using powerbi-api package.
